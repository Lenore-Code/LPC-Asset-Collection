# Breaking changes

There should be very little breaking changes, because assets are only supposed to be added and not to be modified. Some rare breaking changes will be unavoidable though. The file  `versions.json` is a complete history of LPC Asset Collection version numbers and the version that introduced the last breaking change). The list below will explain what changed in those versions and especially *which files were affected*.

 - 0.0.3 : 
	 - Affected files: `/tilesets/plants/harvestable`
	 - Change: Position of the tiles in the graphics changed
